A Wordpress Plugin that Simplifies Process Of Editing Product's Prices in Woocommerce

== Description ==

**MQ Woocommerce Products Price Bulk Edit** lets you to edit Woocommerce Product Prices via 
a simple Admin Page inside Woocommerce Admin Menu. 
This Plugin Uses Ajax that means A Fast and Easy Experience!

####Shortcode
This Plugin Provides A Simple Shortcode for listing all products grouped by their categories. use '[mq_wpbe]' to use this feature.

== Installation ==

for installing every Wordpress plugin you should do these following steps:

1. Decompress the package you downloaded and upload it to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use The Plugin!
